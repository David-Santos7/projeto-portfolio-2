# 🧛‍♂️ Castlevania - Site Estilo Wikipédia 

Arquivo disponível em: 

Este projeto é uma página web desenvolvida por **David Santos Souza**, como uma **reprodução inspirada na Wikipédia**, utilizando a famosa franquia **Castlevania** como base de conteúdo.

Criado inteiramente com **HTML5** e **CSS3**, o site apresenta um layout organizado e informativo sobre a série animada Castlevania (Netflix), além de abordar jogos clássicos da franquia e oferecer navegação entre conteúdos relacionados, como elenco, episódios e produção.

---

## 📌 Objetivo

O principal objetivo deste projeto é **reproduzir o estilo de organização e navegação da Wikipédia**, adaptando seu formato para apresentar conteúdo sobre **Castlevania**. A iniciativa visa praticar e demonstrar habilidades em HTML e CSS puro, aplicadas em um site temático com layout funcional e conteúdo relevante.

---

## 🧱 Estrutura do Site

A página principal é dividida em três áreas principais usando **CSS Grid**:

- `sidebar` — Lado esquerdo com links para os jogos da franquia e imagem dos personagens;
- `main` — Conteúdo principal sobre a série animada da Netflix;
- `anchors` — Lado direito com links internos para outras páginas do site, além de um vídeo incorporado do YouTube.

Além disso, há uma barra de navegação superior com links externos e um rodapé com créditos e direitos autorais.
